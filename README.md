# NovaMind-Ai-Coustom-ChatBot--By-Umar-Imam
NovaMind AI is a multimodal AI assistant built with Streamlit and LangChain, powered by Google Gemini. It supports real-time chat, image analysis, prompt-based reasoning, text-to-speech, chat history export, and session memory, delivering a polished, production-ready AI experience.
